# ESP32-temp-humidity-data-logger
##About this project
This project uses an ESP32 microcontroller, a DHT11 temperature/humidity sensor and Google Firebase Realtime Database to build a simple IoT data logger. It reads environmental data from the sensor every 10 seconds, timestamps it with Unix time and uploads it to Firebase.

##Hardware Required:

*ESP32 Development Board

*DHT11 Temperature & Humidity Sensor module

*Jumper Wires

*Breadboard
